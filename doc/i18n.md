# i18n (Idiomas)

## Como funciona

O Quillit carrega um dicionário interno e expõe uma função `t(key, params, fallback)`.

### Seleção do idioma

- `autoLanguage: true`: tenta usar `document.documentElement.lang` e fallback para `navigator.language`.
- Caso contrário, usa `language`.

## Idiomas disponíveis

Atualmente:

- `pt-BR`
- `en-US`
- `es-ES`
- `fr-FR`
- `de-DE`
- `zh-CN`

## Placeholder padrão

Quando `placeholder` não é passado no `init`, o Quillit usa:

- `placeholder.default` do idioma ativo.

## Chaves principais

- `placeholder.default`
- `tool.*` (labels de ferramentas)
- `toolbar.group.*` (labels de dropdown)
- `modal.*` (títulos e campos de modais)
- `status.*`
- `unit.*` (pluralização)

Observação: se você criar um dropdown custom com `labelKey` (por exemplo `toolbar.group.technical`), lembre de adicionar essa chave em todos os arquivos em `lang/`.

## Exemplos

```html
<html lang="en-US">
```

```js
initQuillit('#editor', {
  autoLanguage: true,
  // sem placeholder: usa o do idioma
});

initQuillit('#editor', {
  language: 'es-ES',
  autoLanguage: false
});
```
